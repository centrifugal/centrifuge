Map presence — massive demo

A single browser tab subscribes to a map_clients presence channel and sees the entire population synchronized over the protocol. Server pushes synthetic entries directly via the broker — no real WebSocket clients are used to generate load.
100,000
Pegboard view — 320 × 320 grid, 2 px cell + 1 px gap. Each member is a distinct rectangle; cell color shows occupancy.
1,000,000
Heatmap view — 1024 × 1024 grid, one pixel per member. At this density individual cells aren't visually separable; bulk fill via canvas ImageData for performance.
