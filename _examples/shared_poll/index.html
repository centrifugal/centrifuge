Security Model

HMAC Signatures Every track request carries an HMAC-SHA256 signature issued by the backend. The server verifies it before allowing access — forged signatures are rejected.
Role-Based Access The role is embedded in the signature. Viewers only receive public flags; admin-only flags are filtered out at signature issuance and double-checked on the server.
Short-Lived Tokens Signatures expire after 30s. The client automatically requests fresh signatures, letting the backend revoke access at each renewal.
Shared Polling All clients share a single backend poll per refresh cycle — no per-client polling. Flag toggles use instant notifications for sub-second updates.
