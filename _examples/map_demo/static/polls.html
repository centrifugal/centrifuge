Live Polls

Real-time voting with server-driven polls, bot voters, and automatic rotation. Vote and see results update instantly across all tabs.

PostgreSQL Transactions: Each vote atomically records the voter (preventing double-voting via KeyModeIfNew) and increments the option's score — all within a single PostgreSQL transaction using cf_map_publish. The score field enables ordered retrieval of results. Three channels: poll:meta (question/timing), poll:results (vote counts via score), poll:votes (dedup).
