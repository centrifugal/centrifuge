Stock Tickers

Real-time stock price feed using map subscriptions. Prices update every second with instant state on subscribe and sector-based filtering via TagsFilter.

Map vs Stream: Unlike a stream subscription (where you wait up to 1s for the next publish), a map subscription delivers the full current state immediately on subscribe. Each ticker is a key in the map — a natural fit for key-value data. Use the sector filter to see TagsFilter in action.
